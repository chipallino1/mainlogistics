**Тема моего проекта “Логистика”**

    1.Суть его такова: у нас есть люди(contacts) или  фирмы(firms) которые хотят,
 	доставить свой товар из пукнта А в пункт В. Мое приложение будет предоставлять возможность для поиска компаний который занимаются перевозками,
 	также оно будет позволять определять оптимальный маршрут(самый дешевый, самый быстрый) определять я это буду с помощью решения задачи коммивояжёра.
 	В моем приложении можно создавать фирмы(либо те которые занимаются перевозками, либо те которые ищут тех кто занимаются перевозками),
 	можно создавать конткаты(это пользоваели которые привязаны к определенной фирму и предоставляют ее услуги(у каждой фирмы может много контактов,
 	но контакт может быть привязан к одной фирме, так же имеется возможность создать контакт который не привязан ни к какой фирме, такой конкакт 
	считается пользователем который хочет чтобы его товар доставили.
	
    2.На данный момент пользователи моего приложения могут создавать профиль,
 	по URL получать фирму (http://localhost:8080/profile/firm/itra@itra.itra http://localhost:8080/profile/contact/egor@mail.ru),
 	могут изменять свой профиль,видеть свой профиль http://localhost:8080/profile/contact/me ,http://localhost:8080/profile/firm/me, 
	пользователи фирмы могут добавлять контакты, просматривать их,удалять.
	
    3.Для запуска моего приложения требуется  в application.properties изменить бд на h2(для вашего же удобства) ,
	 после этого вы можете запускать приложение для того чтобы увидеть работающую часть приложение (а не статические страницы,
 	хотя можете и их посмотреть там картинки котиков) нужно перести по урлу http://localhost:8080/auth далее выбрать кого вы будете создавать
 	или юзер или фирму если вы выбрали юзер то вам следует выбрать либо симпл либо контакт , если вы выбрали контакт то вы сможете привязать свой аккаунт
 	к какой либо фирме если при вводе в инпут автокомплит не срабатывает - следовательно еще не добавлены фирмы либо такой фирмы нет, ваш контакт станет
